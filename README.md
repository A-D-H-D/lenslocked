# Lenslocked

A photo sharing web application built with Go - currently in development.

## 🚧 Project Status: In Progress

I'm working through a comprehensive Go web development course, and this repository 
contains my progress as I build the application step by step.

## ✅ What's Been Implemented So Far

- Basic web server setup with Chi router
- Multiple routes (home, contact, FAQ pages)
- HTML template rendering
- Project structure with separate template files
- Development workflow setup

## 🔜 Coming Up 

- User authentication system
- Database integration (PostgreSQL)
- Photo upload functionality
- User galleries and sharing
- Production deployment with HTTPS
- And much more!

## 🛠️ Current Tech Stack

- **Backend**: Go with Chi router
- **Templates**: Go's html/template package
- **Tools**: Modd for live reload during development

## 📚 About This Project

The goal is to create a production-ready photo sharing application while learning 
real-world development practices including security, database design, and deployment.
